# weather_alert_bot



Bot de alerta de previsão do tempo via Telegram.  

### Funcionalidades
- Consulta a previsão atual usando WeatherAPI.
- Envia mensagem no Telegram com:
  - Condição do tempo
  - Temperatura
  - Umidade
- Agendado para rodar a cada 3 horas.
- Permite execução manual via GitHub Actions.
